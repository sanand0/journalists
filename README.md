# Can AI journalists publish?

- [Hack of the Day](https://sanand0.github.io/journalists/hackoftheday/): A gallery of practical digital-safety tip cards ([docs](hackoftheday/README.md)).\
  Each card is a Mustache-rendered HTML template driven by a JSON file; the gallery is a vanilla-JS page that scales cards into thumbnails, opens a navigable popup, and lets you download cards as HTML or a ZIP.
